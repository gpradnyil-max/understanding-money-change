# Money Magic! - Learn Pounds & Pence

A fun, interactive web app to help kids learn about British money (pounds and pence).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Games

- **Coin Explorer** - Tap coins to learn their names, values, and fun facts
- **Piggy Bank** - Add coins and practise counting up totals
- **Shop & Change** - Buy items and work out the correct change
- **Quiz Challenge** - Timed multiple-choice quiz with stars and scoring
